# SurabayaJS Event Discord Bot

**UnOfficial Bot Discord for Tracking new events in SurabayaJS**

# Setup

- `install nodejs with latest or recommended version from nodejs.org`
- `clone this repo`
- `install all module in package.json`
- `make file .env, and type TOKEN=SECRET_TOKEN`
- `don't forget install dotenv`

## Note for Windows Configuration

- `Before install enmap, and better-sqlite3. Read enmap installation documentaion instead!`
